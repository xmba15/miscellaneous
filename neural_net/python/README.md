## Simple Classification Neural Network Trained on Iris Dataset

# Requirements
 - Use python3 with your pyenv or virtualenv
 - pip install -r requirements

# How to run
 - Training
 ```bash
 python train.py
 ```

# Network Design
 - Optimizer: SGD
 - batch size: 5
 - lr: 0.01
 - Iris Dataset is seperated into (train: 0.8, test 0.2)
